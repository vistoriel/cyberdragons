# Мудрість (Wisdom)

**Мудрість** вимірює інтуїцію, спостережливість та здатність до мудрих рішень. Вища **Мудрість** допомагає уникати пасток, зберігати внутрішній спокій у небезпечних ситуаціях, розпізнавати обман та передбачати наміри інших персонажів.
